# WC-Bet-Advisor 文档目录

本目录包含项目核心文档。

## 文件清单

- **[PRD.md](./PRD.md)** — 开发需求文档(Product Requirements Document)
  - 10 个章节:项目概述、目标用户、功能需求、用户故事、业务流程、数据需求、非功能需求、风险与约束、验收标准、里程碑
  - 用途:产品/项目层面的"做什么"基线

- **[DESIGN.md](./DESIGN.md)** — 详细设计文档(Technical Design)
  - 12 个章节:架构总览、技术选型、数据模型、API 设计、关键算法、前端设计、截止时间模块、部署架构、测试策略、开发规范、风险与缓解、交付物
  - 用途:工程实现层面的"怎么做"基线

## 文档关系

`
PRD.md  ──  "做什么 + 为什么"
   ↓
DESIGN.md ──  "怎么做 + 用什么"
`
